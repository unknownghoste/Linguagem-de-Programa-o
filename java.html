História do Java


O Java é uma linguagem de progamação orientada a objetos que começou a ser criada em 1991 na Sun Microsystems. Teve inicio com o Green Project, que tinha o intuito de antecipar a "proxima onda" que aconteceria na área da informática e progamação. Os criadores do projeto acreditavam que em pouco tempo os aparelhos domésticos e os computadores teriam uma ligação. A primeira invenção dessa equipe era um contole remoto com interface touchscreen, e pra isso foi criado uma linguagem para executa-lo, que era chamada de Oak, o nome de uma árvore. Nesta época estava ganhando sucesso e a equipe do Green Project começou a pensar em aplicações para o Oak na internet. Eles conseguiram adaptar a linguagem Oak para internet, e em 1995 foi criado o Java, que era uma versão atualizada do Oak para internet. A linguagem desenvolvida pelo grupo foi sendo reconhecida, e grandes empresas ofereceram suporte ao Java, para que ele fosse ultilizado em aplicativos de empresas como a IBM. Visto que várias empresas já adotavam a linguagem, o Java se tornou a linguagem de progamação mais usada na história da informática.
